API Documentation
=================

ALBM is written in Fortran 90. Inline source-code documentation is
provided via Doxygen-style comments throughout the source files.

To generate the API documentation locally, install Doxygen
(https://www.doxygen.nl/) and run it from the repository root:

.. code:: shell

   doxygen Docs/Doxyfile

The generated HTML pages will be placed in ``Docs/doxygen_output/html/``
and can be opened in any web browser.
