Advanced Lake Biogeochemistry Model (ALBM)
------------------------------------------

**ALBM** is a one-dimensional, process-based lake biogeochemistry model
developed by Zeli Tan to predict the response of lake thermal and carbon
dynamics and water quality to environmental changes. It consists of
modules for radiative transfer, water/sediment thermal circulation,
water/sediment biogeochemistry, phytoplankton biomass, and gas diffusive
and ebullition transportation.

The source code repository is hosted on
`GitHub <https://github.com/tanzeli1982/ALBM>`_.
Questions and issues can be directed to the developers by opening an
Issue on the GitHub page.

ALBM is a member model of the Inter-Sectoral Impact Model Intercomparison
Project (ISIMIP) lake sector
(`https://www.isimip.org/impactmodels/details/232/ <https://www.isimip.org/impactmodels/details/232/>`_).
